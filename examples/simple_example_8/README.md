# Simple Example

Compute VM Module을 사용하는 예시를 보여줍니다. 

## Usage

- simple_example Designated Directory에는 아래 작업이 구현되어 있습니다.
  - Instance Template 1EA 생성
  - Instance Template 옵션
    - 사설/외부 IP, alias IP Range, 네트워크 태그, 라벨, 삭제 보호,metadata(OSLogin, OSLogin2fa) 등이 설정 되어 있음
    - Boot Disk(IMAGE) 
      - Boot Disk의 source_type을 IMAGE를 통해 배포되도록 설정 되어 있음
      - OS(Ubuntu 20.04 LTS)
    - Attached Disk(1)
      - Regional Disk이며, pd-ssd(Type)및 SNAPSHOT으로 생성되도록 설정 되어 있음
    - Machine Type은 n1-standard-4 설정 되어 있음
    - Service Account의 경우, 기존에 있는 Service Account Attached 하는 형태로 설정 되어 있음
    - Access Scopes은 ["cloud-platform"]으로 설정 되어 있음

- `terraform apply` to apply the infrastructure build
- `terraform destroy` to destroy the built infrastructure
